# EthosAlarmBot

EthosAlarmBot is a Telegram bot for monitoring Ethos Network activity.

The bot helps users track XP, Rank, Contribution Streak, and profile activity directly from Telegram.

## Features

* XP monitoring
* Rank tracking
* Contribution Streak tracking
* Daily reports
* Activity notifications
* Secure account linking
* Optimized Ethos API integration

## Technology Stack

* TypeScript
* Telegraf
* PostgreSQL
* Ethos API

## Purpose

The project was created to help active Ethos community members stay informed without constantly checking the website.
